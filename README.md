# Multi-Crop Plant Disease Detection

This project focuses on deep learning-based precision multi-crop plant disease classification for smart farming in Bangladesh.

## Dataset
- Saon110/bd-crop-vegetable-plant-disease-dataset

## Models
- MobileNetV2
- ResNet50
- EfficientNetV2-S

## Goals
- Multi-crop disease classification
- Explainability using Grad-CAM
- Lightweight deployment
